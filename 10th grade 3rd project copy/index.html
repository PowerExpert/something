How This Page Was Made

1. Creating the iframe viewer

The iframe displays example pages inside the main page.

<iframe name="viewer"></iframe>

The links use target="viewer" so they open inside the iframe.

<a href="example_table.html" target="viewer">Table</a>

2. Creating the menu

The menu is just a list of links.

<ul>
<li><a href="example_text.html" target="viewer">Text</a></li>
<li><a href="example_image.html" target="viewer">Image</a></li>
<li><a href="example_table.html" target="viewer">Table</a></li>
</ul>

3. Using a table for layout

The page layout is made using a table with two columns.

<table>
<tr>

<td>Menu</td>
<td>Iframe Viewer</td>

</tr>
</table>

4. Example of an Image

5. Example of a Video

</video>

6. Example of Audio

</audio>
